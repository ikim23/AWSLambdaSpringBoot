# AWS Lambda: Spring Boot

Simple Spring Boot [Pet Store](https://github.com/awslabs/aws-serverless-java-container/tree/master/samples/springboot/pet-store) project created by [awslabs](https://github.com/awslabs).

Added Serverless framework for seamless deployment.

## Installation

Install required Node.js modules:

```
npm install
```

## Deploy

Run command:

```
serverless deploy
```

Before serverless deploy will be triggered `mvn package` command.
